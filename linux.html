Meet Tux, the Linux mascot

Why Tux, why a penguin?

  • Linus Torvalds likes penguins.
  • He was bitten by one and he fell in love with them.
  • The name was invented by James Hughes and stands for Torvalds's UniX.
  • Created by Larry Ewing in 1996.

Brief history of Linux

  • Unix (1969)
  • GNU (1983)
  • Minix (1989)
  • Linux (1992)

Unix

  • invented at AT&T's Bell Labs (USA)
  • originally with open code
  • later proprietary

GNU

  • Richard Stallman
  • GNU is Not Unix
  • Unix-like Operating System publicly available
  • General Public License (GPL)
  • Incomplete kernel

Minix

  • By Andrew Stuart Tannenbaum
  • Free UNIX-like operating system
  • Created for teaching purposes

Linux

  • By Linus Torvalds from Finland
  • Free UNIX-like kernel written and compiled on Minix
  • Created as a part of university project

Free Software

Free software must fulfill four freedoms:

  1. to run the program for any purpose.
  2. to study how the program works, and change it so it does your computing as you wish.
  3. to redistribute copies so you can help your neighbor.
  4. to distribute copies of your modified versions to others.

Open Source

Open Source is a cover term for software that is defined by a license that makes it open.

Open Source is a concrete way to support the thoughts of Free Software in real life.

Open Licenses

  • GPL
  • BSD
  • MIT
  • Creative Commons
  • others

GPL

You may copy, distribute and modify the software as long as you track changes or dates in source files. Any modifications to or software including (via compiler) GPL-licensed code must also be made available under the GPL along with build and install instructions.

BSD

The BSD 3-clause license allows you almost unlimited freedom with the software so long as you include the BSD copyright and license notice in it (found in Fulltext).

MIT

A short, permissive software license. Basically, you can do whatever you want as long as you include the original copyright and license notice in any copy of the software/source. There are many variations of this license in use.

Creative Commons

  • Attribution (BY)
  • Share Alike (SA)
  • No Derivatives (ND)
  • Non Commercial (NC)

The Creative Commons (BY SA) is a copyleft license.

Advantages of sharing

  • Less work
  • More people to spot errors
  • Better solutions
  • No vendor lock-in
  • Standards
  • No hidden tricks and backdoors
